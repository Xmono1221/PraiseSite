# 褒めてくれるサイト

にじさんじ所属の月ノ美兎さんの「【検証】文字通り『三日三晩』悩んでみたら答えは出るのか！？【後輩を救え！】」の
褒めてくれるサイトが面白かったので作りました。


入力した文章に指定したキーワードが含まれているかを判定し、
結果に応じた画像を表示します。

---

# 注意事項

本アプリはブラウザ内のみで動作します。

サーバーへの送信は行いません。

設定は使用しているブラウザごとに保存されます。

別のブラウザ・別のPCには引き継がれません。

# 使用方法

## 1. 設定

「設定」を開きます。

以下を設定してください。

- 判定したい文字
- 入力画面画像
- 一致画像
- 不一致画像

「保存」を押すと設定がブラウザへ保存されます。

---

## 2. 入力

入力画面で文章を入力します。

入力すると「報告する」ボタンが押せるようになります。

---

## 3. 判定

入力内容に設定した文字が含まれているか判定します。

英字は大文字・小文字を区別しません。

例

```
設定文字

Kou
```

以下はすべて一致します。

```
kou
KOU
Kou
```

---

# 設定について

設定はブラウザのLocal Storageへ保存されます。

保存される内容

- 判定文字
- 入力画面画像
- 一致画像
- 不一致画像

ブラウザのデータを削除すると設定も削除されます。

---

# 画像サイズ

画像は Base64 へ変換して保存しています。

推奨サイズ

```
500KB以下
```

500KBを超える画像も保存できますが、
ブラウザのLocal Storage容量（約5MB）を超えると保存できません。

保存できない場合は容量超過メッセージが表示されます。

---

# 設定初期化

設定画面の

```
設定をクリア
```

を押すと

- 判定文字
- 入力画像
- 一致画像
- 不一致画像

を削除します。
